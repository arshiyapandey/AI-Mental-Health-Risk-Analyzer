# AI Mental Health Risk Analyzer

## Overview
This project uses machine learning to analyze factors influencing mental health risk using a large mental health survey dataset (~292k responses).

## ML Pipeline
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Train/Test Split
- Model Training (Logistic Regression, Decision Tree, Random Forest)
- Model Comparison
- Feature Importance Analysis

## Best Model
Decision Tree (Accuracy ≈ 78%)

## Key Influencing Factors
- family_history
- care_options
- Country
- Gender
- mental_health_interview
